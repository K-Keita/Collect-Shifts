## 使用技術
- React.js(react-hooks)
- Redux
- firebase

  - ホスティング
- material-ui

## 機能一覧
- アカウント登録、ログイン機能
- シフトの登録、変更機能
- グループ機能
  - グループの新規作成
  - 既存グループへの参加
