# Collect Shifts
一週間ごとのシフトを集め、管理するアプリです。実際にシフト集めの際に、使用してもらっています。

## 使用技術
- React.js(react-hooks)
- Redux
- firebase
  - 認証
  - データベース
  - ストレージ
  - デプロイ
- material-ui

## 機能一覧
- アカウント登録、ログイン機能
- シフトの登録、変更機能
- グループ機能
  - グループの新規作成
  - 既存グループへの参加
